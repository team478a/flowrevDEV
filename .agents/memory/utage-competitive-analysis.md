# UTAGE 競合分析

## UTAGE 概要
- URL: https://utage-system.com/
- ポジション: 日本向けオールインワン集客・販売自動化 SaaS（直販型）
- 料金: ライト ¥9,700/月 / スタンダード ¥21,670/月（税込）、14日間無料トライアル

## UTAGE の主要機能
- LP・ファネル作成（スタンダード：無制限）
- メール配信 ＋ **LINE配信**（両方）— FlowRev との大きな差
- 会員サイト・コース管理（スタンダード：無制限、動画1TB）
- 決済（一括・分割・オーダーバンプ）
- アフィリエイト機能（スタンダードのみ）
- オペレーター最大5名
- AIアシスト（簡易）

## FlowRev の優位点（UTAGEにない機能）
1. **ホワイトラベル（OEM）** — UTAGEに完全不在。最大の差別化ポイント
2. **4階層マルチテナント** — コンサルが複数クライアントを一元管理できる
3. **AI統合の深さ** — Claude/GPT でLP・商品・フォローメール生成（UTAGEは簡易）
4. **代理店ビジネスモデル対応** — UTAGEでは実現不可

## FlowRev の劣位点（今後の実装課題）
| 優先度 | 機能 | 備考 |
|--------|------|------|
| 🔴 高 | **Stripe決済** | 基盤実装済み、UIを完成させる（Task #1）|
| 🔴 高 | **LINE Messaging** | 日本市場必須（Task #2） |
| 🟡 中 | **動画ホスティング** | Cloudflare Stream（Task #3） |
| 🟡 中 | アフィリエイト機能 | 将来フェーズ |
| 🟡 中 | LP テンプレート | UTAGEは豊富 |

## 戦略的ポジショニング
- UTAGE：「1人事業主が自分のビジネスを回す」ツール
- FlowRev：「コンサル・代理店が複数クライアントを自社ブランドで管理する」ツール
- ターゲット：マーケコンサル・スクール運営・複数ブランド事業者・代理店

**Why:** 競合が存在しないホワイトラベル領域に集中することでニッチ優位を取る。
**How to apply:** 新機能を追加する際は「ホワイトラベル文脈で意味があるか」を確認。
